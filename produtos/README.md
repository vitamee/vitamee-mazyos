# Produtos

Catálogo da Vitamee — fichas técnicas, benefícios, fotos e copy de venda por produto, além de precificação.

## Como organizar
- Um arquivo ou subpasta por produto (ex: `magnesio-quelato/`, `hair-skin-gummies/`)
- `catalogo-geral.md` — visão consolidada de toda a linha
- `precificacao.md` — custo, preço, margem por produto
